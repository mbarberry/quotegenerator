# Quote Generator

## Overview

NodeJS server deployed on AWS as a Lambda function with CloudFormation and a HTML custom element frontend that allows users to interact with quotes from popular TV shows.

## Features

- **AWS Lambda Backend** – Serverless NodeJS functions handle API requests.
- **CloudFormation Deployment** – Infrastructure managed and deployed with AWS CloudFormation.
- **HTML Custom Element Frontend** – Interactive components for displaying TV show quotes.
- **User Interaction** – Easily explore and generate quotes from popular shows.
